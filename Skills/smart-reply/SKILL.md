# 智能回复 (Smart Reply)

## 描述
智能回复抖音私信，根据消息上下文生成个性化回复。

## 触发条件
- 关键词: ["回复消息", "智能回复", "帮我回复"]
- 意图: message_reply
- 事件: 收到新消息时自动触发 (可配置)

## 依赖工具
- `get_messages`: 获取消息
- `screenshot`: 页面截图
- `send_message`: 发送消息

## 工作流程

### Step 1: 获取消息上下文
```
动作: 获取与好友的最近消息
工具: get_messages(friend_id, limit=10)
输出: messages[]
```

### Step 2: 分析消息意图
```
动作: 使用 AI 分析消息
输入: 消息历史
分析:
  - 消息类型: 问候/问题/分享/其他
  - 情感倾向: 积极/中性/消极
  - 是否需要回复: 是/否
输出: intent_analysis
```

### Step 3: 生成回复
```
动作: 根据分析生成回复
规则:
  - 回复要自然、友好
  - 如果是问题，给出有意义的回答
  - 适当使用 emoji
  - 长度: 5-100 字 (根据对话)
```

### Step 4: 发送回复
```
动作: 发送生成的回复
工具: send_message(friend_id, reply)
```

## 回复策略

### 问候类
```
输入: "你好", "在吗", "早啊"
回复示例:
  - "在呀！有什么事吗？😊"
  - "早呀！今天怎么样？"
  - "你好~ 有什么可以帮你的？"
```

### 问题类
```
输入: "最近在忙什么？"
回复示例:
  - "最近在工作/学习呢，你呢？"
  - "在忙一些项目，有点累但挺充实的"

输入: "周末有什么计划？"
回复示例:
  - "打算休息一下，可能去看电影~ 你呢？"
  - "还没想好，可能宅家吧哈哈"
```

### 分享类
```
输入: [分享视频/图片]
回复示例:
  - "哇好厉害！👍"
  - "这个好有意思！"
  - "哈哈笑死我了 😂"
```

### 表情包类
```
输入: [表情包]
回复示例:
  - [回复表情包]
  - "哈哈这个表情好逗 😂"
```

## 个性化配置

### 回复风格
- `friendly`: 友好热情 (默认)
- `casual`: 随意轻松
- `formal`: 正式礼貌
- `humorous`: 幽默风趣

### 特殊好友规则
```yaml
special_rules:
  - name: "老板"
    style: formal
    keywords: ["工作", "项目", "会议"]

  - name: "闺蜜"
    style: casual
    use_emoji: true
    use_sticker: true
```

## 注意事项

1. **回复时机**
   - 不要秒回 (显得太闲)
   - 建议延迟 30秒-5分钟

2. **回复频率**
   - 不要每条都回
   - 适当"漏掉"一些消息

3. **内容安全**
   - 不发送敏感内容
   - 不承诺任何事情
   - 不泄露隐私信息

4. **人工接管**
   - 复杂对话通知用户
   - 重要消息标记需人工处理

## 示例对话

```
[收到新消息]
好友: 好友A
消息: "最近怎么样啊？好久没聊了"

[AI 分析]
类型: 问候+关心
情感: 积极
需要回复: 是

[生成回复]
"挺好的！最近在忙一些事情，你呢？好久不见~"

[发送成功]
```

## 配置选项

```yaml
smart_reply:
  enabled: true
  auto_reply: false        # 是否自动回复
  reply_delay_min: 30      # 回复延迟最小值 (秒)
  reply_delay_max: 300     # 回复延迟最大值 (秒)

  style: friendly          # 默认回复风格
  use_emoji: true          # 使用 emoji
  use_sticker: false       # 使用表情包

  notify_user: true        # 通知用户有新消息
  require_confirmation: false  # 发送前需要确认

  blacklist:               # 不自动回复的好友
    - "好友X"

  keywords_ignore:         # 忽略的关键词
    - "群发"
    - "广告"
```
